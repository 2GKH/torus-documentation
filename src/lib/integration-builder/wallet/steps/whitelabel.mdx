---
title: Whitelabel/customize brand logo and colors (optional)
---

You can also whitelabel or customize logo and colors to show your brand in the
Wallet:

```js
await torus.init({
  whitelabel: {
    theme: {
      isDark: false,
      colors: {
        torusBrand1: "#000000",
        torusGray2: "#FBF7F3",
      },
    },
    logoDark: "https://startrail.io/images/front/startrail-top__main.svg", // Dark logo for light background
    logoLight: "https://images.toruswallet.io/startrail-logo-light.svg", // Light logo for dark background
  },
  // ...
});
```

See [API reference](/wallet/api-reference/class#whitelabel) for more details.
