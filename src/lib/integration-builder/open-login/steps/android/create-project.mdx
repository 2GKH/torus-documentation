---
title: Create OpenLogin project
---

Go to [Developer Dashboard](https://developer.tor.us), create or select an OpenLogin project:

- Add `{YOUR_APP_PACKAGE_NAME}://auth` to **Whitelist URLs**.

- Copy the Project ID for usage later.
